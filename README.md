# YouTube Revenue Optimization Case Study

## Problem Statement
How can YouTube increase its revenue?

## Approach
- Built a KPI tree to break down revenue into key drivers
- Identified leading product metrics like CTR, engagement, and retention
- Mapped user journey from visitor to viewer to engagement to monetization

## Key Metrics
- Video Thumbnail CTR
- Bounce Rate
- Average Video Plays per User
- Video Completion Rate

## Solution
Proposed actionable strategies:
- Improve thumbnails and recommendations to increase CTR
- Reduce bounce rate through better personalization
- Increase watch time using autoplay and playlists



## Outcome
Demonstrates product thinking, metric breakdown, and data-driven decision making for revenue growth.
